# foo i18n

Type-safe localisation.

Read the documentation for more information. _(Coming soon!)_

## Development

This project is using [Turborepo ](https://turbo.build/repo) and [pnpm](https://pnpm.io/).

### Install everything

```
pnpm i
```

### Reset all dependencies

```
pnpm reset
```
